# 📊 API Data Dashboard

A clean, responsive web dashboard that **fetches live data from any REST API** and displays it in interactive charts, tables, and filters.
Updates in real time, supports instant search, and works on any device.

> Helped a client monitor live data in one clean view — instead of digging through raw API responses.

---

## ✨ Features

- ✅ Fetches live data from any REST API
- ✅ Interactive charts (powered by Chart.js)
- ✅ Summary stat cards (totals at a glance)
- ✅ Real-time search & filtering
- ✅ Clean, sortable data table
- ✅ Fully responsive (mobile-friendly)
- ✅ No backend needed — pure frontend

---

## 🛠️ Tech Stack

- **HTML5** – structure
- **CSS3** – modern responsive design
- **JavaScript (ES6)** – logic & API calls
- **Chart.js** – data visualization
- **Fetch API** – live data loading

---

## 🚀 How to Run

No installation needed — it's a pure frontend project.

```bash
# 1. Clone the repo
git clone https://github.com/waseemqadir345/api-data-dashboard.git
cd api-data-dashboard
```
---

⚙️ Use Your Own API
Open script.js and change one line:

---

const API_URL = "https://your-api-endpoint.com/data";

---

Then adjust the table columns and chart fields to match your API's data structure.
